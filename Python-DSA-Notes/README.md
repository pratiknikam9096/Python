# Python-DSA-Notes

![python](https://img.shields.io/badge/Python-3.8%2B-blue)
![repo](https://img.shields.io/badge/Repo-Educational-orange)

Comprehensive, beginner-friendly notes and solved problems for Python fundamentals and Data Structures & Algorithms (DSA). This repository is designed for students, interview preparation, and self-learners.

## Learning Roadmap

- Start with `Python_Fundamentals` to learn core language features.
- Practice problems in `Python_Fundamentals/practice_problems.md`.
- Move to `DSA` folder and follow topics in order: Arrays → Strings → Linked_List → Stack → Queue → Recursion → Searching → Sorting → Hashing → Trees → Graphs → Dynamic_Programming.

## Folder Structure

See the repository root structure:

```
Python-DSA-Notes
┣ Python_Fundamentals
┣ DSA
┗ README.md
```

Detailed folder-level READMEs explain each topic.

## How to use

1. Clone the repo:

```bash
git clone <repo-url>
cd Python-DSA-Notes
```

2. Read fundamentals, try practice problems, then move to DSA topics.

- Beginners learning Python
