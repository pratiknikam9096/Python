# Python Fundamentals

This folder contains beginner-friendly notes for core Python concepts, examples, best practices, common mistakes, and practice problems with solutions.

Files:

- `control_flow.md` — conditionals and loops
- `functions.md` — functions, scope, lambdas
- `strings.md` — string operations and methods
- `lists_tuples_sets.md` — collections overview
- `dictionaries.md` — mapping type and usage
- `oops.md` — OOP concepts in Python
- `exception_handling.md` — try/except/finally
- `file_handling.md` — reading/writing files
- `practice_problems.md` — 20 practice problems with solutions

Start with `basics.md` and work through the examples.
