# Queue

Definition: FIFO (first-in-first-out) structure.

Why used: Scheduling, BFS, buffering.

Complexities:

- Enqueue/Dequeue: O(1), Space: O(n)

Real-world: People standing in a line.

Diagram:

Front -> [a] [b] [c] <- Rear

Advantages: Fair processing order.
Disadvantages: Limited access to interior elements.
